# CSV Analyzer - AI Dataset Storyteller

A beautiful, interactive web application that analyzes CSV files and generates AI-powered insights, visualizations, and narratives.

## Features

- 📊 **Drag & Drop CSV Upload** - Easy file upload interface
- 🤖 **AI-Powered Analysis** - Generates compelling data stories using Anthropic's Claude AI
- 📈 **Automatic Visualizations** - Creates charts and graphs automatically
- 💡 **Smart Insights** - Identifies patterns and trends in your data
- 🎨 **Beautiful UI** - Modern, elegant design with smooth animations

## How to Use

1. Open `CSV_Analyzer.html` in your web browser
2. Drag and drop a CSV file or click to browse
3. Enter your Anthropic API key when prompted (or click "API Key" button in header)
4. View your data story with AI-generated insights and visualizations

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources and API calls)
- Anthropic API key (get one at https://console.anthropic.com/)

## Technologies Used

- HTML5, CSS3, JavaScript
- Chart.js for visualizations
- PapaParse for CSV parsing
- Anthropic Claude AI for data analysis

## Setup

1. Clone this repository
2. Open `CSV_Analyzer.html` in your browser
3. Configure your API key using the "API Key" button in the header
